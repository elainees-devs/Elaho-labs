# Elaho Labs Business Website

Welcome to the Elaho Labs business website project! This repository contains the source code for the official website of Elaho Labs, a company dedicated to building modern web, mobile, and AI solutions.

## Features
- Responsive landing page
- Contact form
- Service and project showcase
- Modern UI with Tailwind CSS

## Tech Stack
- React (TypeScript)
- Vite
- Tailwind CSS
- Lucide Icons

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/elainees-devs/Elaho-labs.git
   cd Elaho-labs
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

The site will be available at `http://localhost:5173` by default.

## Folder Structure
- `src/components/` – UI components and sections
- `src/pages/` – Page components
- `public/` – Static assets
